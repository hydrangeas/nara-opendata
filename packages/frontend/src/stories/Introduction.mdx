## Components

### Core Components

- **Button**: Various styles and states for user actions
- **Card**: Container component for content grouping
- **Input**: Form input with validation support
- **Modal**: Overlay dialog for focused interactions
- **Table**: Data display with sorting and selection

### Design Principles

1. **Consistency**: Uniform spacing, colors, and interactions
2. **Simplicity**: Clean interfaces without unnecessary complexity
3. **Flexibility**: Components that adapt to various use cases
4. **Accessibility**: Inclusive design for all users
5. **Performance**: Minimal bundle size and optimal rendering

## Getting Started

```bash
# Install dependencies
npm install

# Run Storybook
npm run storybook

# Run tests
npm run test

# Build components
npm run build
```

## Usage Example

```tsx
import { Button, Card, CardHeader, CardTitle, CardContent } from '@opendata-api/frontend/components/ui';

function MyComponent() {
  return (
    <Card>
      <CardHeader>
        <CardTitle>Welcome</CardTitle>
      </CardHeader>
      <CardContent>
        <p>Build amazing interfaces with our component library.</p>
        <Button variant="primary">Get Started</Button>
      </CardContent>
    </Card>
  );
}
```
